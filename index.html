External Fonts

Adding external fonts can be done in two ways.

  1. HTML
    • Copy the provided <link> with the href
    • Paste this into the <head> of the html document
  2. CSS
    • Find the @font-family code within the stylesheet (use the href in the provided <link>)
    • Paste this at the top of the CSS sheet
